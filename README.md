--- Desafio de Projeto DIO – O Dia a Dia de um QA
Este repositório contém a documentação completa do desafio de projeto O Dia a Dia de um QA: A Prática de Testes Manuais Funcionais, parte da formação Quality Assurance da DIO
O objetivo do projeto é aplicar os conceitos de testes manuais funcionais em um ambiente simulado de e-commerce (Swag Labs), utilizando técnicas como BDD, análise de requisitos e elaboração de casos de teste.
Todos os documentos foram elaborados em Word e exportados para PDF conforme exigido pelo desafio.

--- Ferramentas Utilizadas
- Microsoft Word (documentação)
- Canva / XMind (mapa mental)
- GitHub (versionamento e entrega)
